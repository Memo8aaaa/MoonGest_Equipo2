Descripción del Proyecto:
MoonGest es un sistema de software de gestión interna desarrollado para Moon Beauty Space, 
un salón de estética especializado en servicios de pestañas.
El sistema surge como respuesta a la problemática operativa actual del negocio, 
donde toda la gestión depende de métodos completamente manuales: una libreta física para registrar citas, 
historial de clientas, pagos y ganancias diarias, y WhatsApp como único canal de coordinación. 
Esta operativa manual ha generado errores como duplicación de citas, pérdida de información, 
dificultad para consultar registros anteriores y una carga administrativa excesiva para la encargada.
MoonGest es una herramienta de escritorio de uso exclusivo para la encargada del salón, que le permitirá digitalizar y 
centralizar toda la operación del negocio en una sola plataforma. Su interfaz está estructurada en cuatro módulos principales 
accesibles desde un menú lateral: Calendario, Clientas, Servicios y Reportes.
El módulo de Calendario de Citas es el núcleo del sistema. Muestra todas las citas del día con indicadores visuales de color 
que reflejan el estado de cada una: rojo para citas sin anticipo, amarillo para anticipo registrado y verde para citas pagadas completamente.
Desde esta vista la encargada puede agendar nuevas citas mediante un formulario en tres pasos
(selección de clienta, selección de servicio y asignación de fecha/hora), consultar el detalle de cada cita con 
información de la clienta, servicio, precio total, anticipo pagado y saldo pendiente, así como reprogramar o cancelar citas 
indicando el motivo correspondiente. El calendario es interactivo y muestra en tiempo real la disponibilidad de horarios, 
utilizando colores verde para horarios disponibles, rojo para horarios ocupados y morado para la hora seleccionada, 
evitando así los conflictos de doble reserva.
El módulo de Clientas permite consultar su historial de servicios anteriores y visualizar alertas sobre alergias relevantes 
(como alergia a látex) directamente en la tarjeta de la clienta, dato crítico para la seguridad del servicio.
El módulo de Servicios administra el catálogo completo del salón, que incluye todos los tipos de pestañas 
(Clásicas $330, Rímel $350, Híbridas $350, Volumen $400, Volumen Tecnológico $410, Mega Volumen $550, Wispy $450) 
con sus respectivos retoques y servicios de Retiro, así como servicios de uñas. La encargada puede agregar nuevos servicios 
indicando nombre, duración y precio en MXN, editar servicios existentes y activar o desactivar servicios del catálogo 
según disponibilidad.
El módulo de Pagos permite registrar el anticipo de $100 pesos requerido para confirmar una cita, adjuntando el comprobante 
de transferencia en formato JPG, PNG o PDF (máximo 5MB) junto con el número de referencia/rastreo. 
Una vez concluido el servicio, la encargada registra el pago final indicando el método utilizado 
(efectivo o transferencia bancaria), con lo que el sistema calcula automáticamente el saldo pendiente, actualiza el estado 
de la cita a "Pagado Completamente" y bloquea la edición de los montos para evitar alteraciones.
El sistema cuenta con un sistema de alertas contextuales que guía a la encargada en todo momento: alertas rojas con iconos 
para errores de validación (como formato de teléfono inválido o campos incompletos), alertas verdes para confirmaciones 
exitosas (como "Cita agendada exitosamente") y alertas con sugerencias cuando se detectan conflictos de horario.
El proyecto se desarrolla bajo la metodología RUP (Rational Unified Process), siguiendo sus fases de Inicio, Elaboración, 
Construcción y Transición, con un enfoque iterativo que permitió validar el prototipo directamente con la encargada del 
negocio antes de su implementación definitiva.
